Bootstrap Electron app

Fast local flow

Single-board workflow with three columns, drag and drop, quick card creation, and local persistence.

Offline-first by design

SQLite is the source of truth. The first milestone works fully locally, without sync, OAuth, or external services.

Built for the desktop

Electron shell, always-on-top support, and a layout designed to stay visible while you work.

Current stack

Pragmatic technology, minimal operational cost.

Stickban is built around Electron, React, TypeScript, SQLite, Zustand, Tailwind CSS, and dnd-kit. The bootstrap milestone avoids remote sync and infrastructure on purpose.

  • Electron + React + TypeScript
  • SQLite via better-sqlite3
  • Zustand for UI state
  • Tailwind CSS styling
  • dnd-kit interactions

Project status

Open-source, AI-assisted, and still moving fast.

Stickban is being built in public, with an AI-assisted workflow and manual review. The public site and desktop app are deployed independently so product releases and project communication can evolve in parallel.
